# Animated Portfolio Website

A personal portfolio website featuring smooth scrolling, interactive animations, and a modern dark-themed aesthetic.

## Features

- **Interactive Animations**: Image sequence animations and glassmorphism UI elements.
- **Responsive Design**: Designed to work across various screen sizes.
- **About Page**: Detailed personal bio and tech stack showcase.
- **Tech Stack**: HTML5, CSS3, JavaScript.

## Setup

1. Clone the repository.
2. Open `index.html` in your web browser.
3. No build tools are required for the front-end code.

## Folder Structure

- `/index.html`: Main portfolio page.
- `/about.html`: About me page.
- `/style.css`: All styles and animations for the website.
- `/script.js`: JavaScript for animations and interactions.
- `/assets`: Images and animation sequences used in the project.
